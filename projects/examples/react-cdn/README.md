# react-cdn

React 19 via CDN with Tailwind CSS. No build step required.

## Preview

Deploy to Vercel and view via the deployment URL:

```bash
./skills/vercel/scripts/setup.sh projects/examples/react-cdn
./skills/vercel/scripts/push.sh projects/examples/react-cdn
```

The deployment URL is returned after push. Use [browserless](../../../browserless/) for visual verification and testing.

## Features

- React 19 from esm.sh
- Tailwind CSS from CDN
- No build tools needed
- Edit → deploy → verify cycle
