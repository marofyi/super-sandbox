# Contributing

Guidelines for contributing to this repository.

## Git Workflow

1. Work on feature branches, never directly on main
2. Write clear, descriptive commit messages
3. Create PRs for human review before merging

## Adding Content

Before adding new skills, tools, or documentation:

1. Check if existing content already covers the topic
2. Follow the existing directory structure
3. Ask before adding if unsure
